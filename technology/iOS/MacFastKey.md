###Mac常用快捷键

###### Mac键盘
>⌘	Command key  
>⌃	Control key  
>⌥	Option key  
>⇧	Shift Key  
>⇪	Caps Lock  
>Fn	Function Key

---
>* option + command + h 最小化除当前的其它所有窗口，准确说是隐藏
* command + m 最小化当前窗口
* command + L 恢复最小化窗口
* command + shift + 3 截全屏
* command + shift + 4 选区域截屏
* command + shift + 4 +空格 截取的窗口
* Control+command+空格键 表情😂
